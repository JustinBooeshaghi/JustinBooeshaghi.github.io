# JustinBooeshaghi.github.io

This is my personal portfolio website, showcasing my projects in physics simulations, mathematics, and cosmological research. As a student at Langley High School, this site serves to attract college recruiters and demonstrate my passion for science and technology.

## Structure

- `index.html`: Main landing page
- `css/style.css`: Stylesheet for the website
- `js/script.js`: JavaScript for interactive elements
- `assets/`: Directory for images and other assets

## Getting Started

To view the website locally:

1. Clone the repository
2. Open `index.html` in your web browser

For GitHub Pages deployment, the site is automatically built from the `main` branch.

## Customization

- Update personal information in `index.html`
- Add your projects in the projects section
- Modify styles in `css/style.css`
- Add images to `assets/` directory

## Technologies Used

- HTML5
- CSS3
- JavaScript
